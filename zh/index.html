日檢N3心得

斷斷續續地準備 最初應該是高一下的時候立志要學日文吧， 說是「立志」好像太勤勉了，其實比較像是一時興起而已。 高二也是斷斷續續地囫圇吞棗地背單字背句型。 然後高三幾乎沒碰過日文，超級怠惰。 真的很難統計自己到底花了多少時間在這個語言上面。 在大一時鬼迷心竅，終於報名了2020/12/06 的日檢N3。 檢定當日 N3, N4, N5 都是分三個部份： 語言知識 讀解 聽解 考試也是依序分三節來考。 當天身體狀況OK，但是考試過程有點太刺激。 語言知識：命運二選一 有不少題目就只是問漢字的羅馬拼音， 或是給羅馬拼音再讓你選搭配的漢字， 母語是中文的話用猜的也會對一半吧，真吃香。 然後因為自己可悲的單字量，必須面對超多的單字二選一， 像是哈欠(あくび)和噴嚏(くしゃみ)稿混了 像是零用錢(こぜに)和找錢(おつり)猜錯了 像是膝蓋(ひざ)和手肘(ひじ)完美地弄反了 剛考完這部份時真的怕爆，不過也沒有時間哀悼就開始讀解測驗了。 讀解：佛系閱覽 就是標準的閱讀測驗，印象中沒有什麼太奇葩的句型。 真的遇到不會的句型就看看前後文猜意思吧。 然後就是速度很重要，不要花太多時間去讀通每個句子。 應該說要先看題目，然後再佛系地掃過文章找答案。 閱讀速度要再加強阿，題目做過一遍時只剩大概10分鐘了。 照著個速度N2、N1一定讀不完。 聽解：音速日語 剛聽到它念例題的時候心臟慢了半拍、臉直接垮掉， 這個速度怎麼比題本和單字書快那麼多？ 然後這個語言念的快的時候又容易糊在一起，光是去聽主詞和動詞就蠻吃力的了。 一臉矇逼的我正在試圖適應這到底什麼語速。 幸好聽力的每小題都蠻短的而且關鍵字的發音都很清楚，再長一點點我就完全跟不上了。 自從本次聽力測驗，背單字時例句都是放1.25倍速了。 早點習慣音速日語吧。 不要輕易相信模擬考題那不急不徐的語速， 不然就會像我一樣在考試當日嚇個半死。 檢定結果 檢定結果…比想像中的好欸？ 然後通知書上還寫說是PR92? 感覺有點莫名奇妙。 看來這次的運氣是真的很好，語言知識應該有不少二選一都猜對了。 聽解應該有不少三選一猜對了，因為考試當日真的是聽的魂不守舍。 不管怎樣過了N3還是蠻高興的，但是接下來的路還很長。 漫長的N2路上 說來慚愧，雖然離考完N3到我寫這份心得已經過了半年了 N2的單字書我才啃到一半而已。 文法書幾乎沒碰過，都是遇到奇妙的文法才去翻一下。 真是一曝十寒的效率，N1更是遙遙無期。

反正就開始寫Blog了

渾渾噩噩、度年如日 每天光是起床都覺得好累，莫名其妙活了十九年卻一年比一年不知道自己再幹麻，但是睡太多又會有罪惡感，不知道從什麼時候開始已經抓不到時間流逝的感覺了。 如果一個學期或一個學年感覺過得那麼快卻沒有鮮明的回憶的話，應該是我的生活方式出了什麼問題吧？每個人醒來時應該都至少要有一件值得期待、嚮往、跳下床去執行的事情吧？ 好希望能夠找回之前那種充滿熱忱、幹勁滿滿地把一件事情做到最好的感覺阿。 所以嘗試去紀錄重要的當下… 為了抓回一點時間感，還是來紀錄一下生活中的趣事好了，以後再來回味的時候就比較不會有虛度光陰的感覺了…吧。 然後不要像我假日直接睡12小時，高速虛度光陰。 但是日記什麼的實在太勤勉了，每天都要找東西寫下來。就算是很無聊的一天也要硬寫，實在太累太麻煩了不太適合我。 果然還是來寫一下Blog好了。 把好玩的、重要的事件按照當下的心情紀錄下來。 那麼該寫些什麼東東好呢 目前想到的可能會紀錄的東西有： 各式突發奇想 (有機會是廢文) Code心得 (專業IT廢文) 美食番 (好吃的廢文？) 廢文 不過最重要的果然還是要有Fu，等到心情正確再執筆就好了。 配置 如果單純寫Blog的話門檻蠻低的，找個Static Site Generator再套個Theme基本上就能看了。 CMS: Hugo Hosting: Github Pages 等到文件量太多了再租個domain搬出去八。 老實說我也不知道自己想做什麼、在做什麼，反正就開始寫Blog了。 在這疫代疫代的時代 恩，最近還挺亂的。缺水缺電缺疫苗，被三級警戒封印在家的各位如果想出門一定要帶口罩喔，口罩要蓋過鼻子喔。 Stay Safe, Stay Negative.
